# Plan: 文明的赓续

## Taxonomy

这个文明给自己立下的那道未竟之任：在重返世界、走向现代的同时，把使它得以延续至今的根本接续下去，续住把万方系于一书的[汉字](piece_hanzi.md)，续住以[孝](position_xiao.md)为本、家国相连的人伦，续住[三教](position_sanjiao.md)涵养出的从容与中和，并在每一次更替之后，依[天命](position_tianming.md)的古老道理重整旗鼓，使断处复续。

## Owner

- Project: khai-cultures

## Direction

变其形以应时世，不失其魂以续文脉：让古老的文字在新的世道里照样书写，让家与国相连的人伦在新的生活里照样有根，让儒道释的中和在新的喧嚣里照样养心；让文明既走得进现代、走得向世界，又始终认得出自己的来处，把"延续"二字，落到每一代人的日用之间。

## Orders

- 对在上者：以民为本，知权柄为托付而非私产，使[天命](position_tianming.md)所系的民心得安，太平得续。
- 对寻常人家如[老百姓](persona_laobaixing.md)：把字传下去、把根记下去、把家立起来，于日用之间续住文脉。
- 对走向世界的华人：无论走得多远，仍以汉字、年节与乡音，把这个文明带到天涯海角而不忘其本。

## Implementation

一个绵延数千年、从未中断的文明，已在由衰而振的百年里重开门户、重整家业，沉稳地走回世界；而更长的一程仍在脚下：要在古老与新潮之间守住中和，在本土与世界之间认清来处，使这条从未断流的长河，既改其道以行远，又不失其源以归宗。

## Targets

- [x] 由甲骨一脉相承的[汉字](piece_hanzi.md)至今未断，仍为亿万人日日书写。
- [x] 以[孝](position_xiao.md)为本、家国相连的人伦，历代相传而未绝。
- [x] 经由衰而振的百年，文明重开门户，沉稳地走回世界。
- [W] 在古老与新潮之间，守住[三教](position_sanjiao.md)所养的从容与中和，不为一时的喧嚣所夺。
- [W] 在重返世界的同时，使文明既走得远、又认得出自己的来处。
- [ ] 让散居五洲的华人世代相传，仍以汉字、年节与乡音不忘其本。
- [ ] 把"变其形而不失其魂"的赓续之道，交到每一代新人的手中。
