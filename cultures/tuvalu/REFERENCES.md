# Tuvalu: References

The provenance behind this culture: the Hofstede-style source data that tunes the
pitch (kept here, never in the prose) and the public-record sources behind the
plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_tvl.md](pitch_tvl.md) is written from Tuvalu's cultural profile in
the spirit of Geert Hofstede's model of national culture and names no dimension and
carries no number. Tuvalu is not separately scored in the published Hofstede data;
the profile below is the qualitative reading used to tune the pitch (a small,
communal, consensus-bound, faith-filled, modest, and resilient ocean society),
recorded here so the reasoning is on the record without entering the prose.

| Dimension                     | Tuvalu profile (qualitative)                              |
| ----------------------------- | --------------------------------------------------------- |
| Power Distance                | moderate to high (respect for elders and the falekaupule) |
| Individualism                 | low (strongly communal, kin- and island-bound)            |
| Motivation toward Achievement | low (modest, consensus over competition)                  |
| Uncertainty Avoidance         | moderate (custom and faith as anchors)                    |
| Long Term Orientation         | reading not separately published                          |
| Indulgence                    | reading not separately published                          |

Source: Geert Hofstede, Hofstede Insights (the six-dimensions model of national
culture). Tuvalu is not separately published in the Country Comparison data; the
profile above is a qualitative reading and is not an official Hofstede score.

## Language note

The prose body of this culture is written in Tuvaluan (te gana Tuvalu, ISO 639-3
tvl), a Polynesian language close to Samoan; only the H2 headings are in English.
The tvl tag is not in franc's detection model, so it is handled via the house's
khai.languages exempt list and an NLP verification layer configured on main.
