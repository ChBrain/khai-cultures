# Finland: References

The provenance behind this culture: the Hofstede source data that tunes the pitch
(kept here, never in the prose) and the public-domain historical sources behind
the plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_fi.md](pitch_fi.md) is written from Finland's profile on Geert
Hofstede's model of national culture and names no dimension and carries no
number. The scores and their source are recorded here so the data is on the
record without entering the prose.

| Dimension                     | Finland score (approx.) |
| ----------------------------- | ----------------------- |
| Power Distance                | 33                      |
| Individualism                 | 63                      |
| Motivation toward Achievement | 26                      |
| Uncertainty Avoidance         | 59                      |
| Long Term Orientation         | 38                      |
| Indulgence                    | 57                      |

Source: Geert Hofstede, Hofstede Insights (the Country Comparison tool and the
six-dimensions model of national culture). Scores are approximate, as published
by Hofstede Insights / The Culture Factor Group.
