# Luxembourg: References

The provenance behind this culture: the Hofstede source data that tunes the pitch
(kept here, never in the prose) and the public-domain historical sources behind the
plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_lu.md](pitch_lu.md) is written from Luxembourg's profile on Geert
Hofstede's model of national culture and names no dimension and carries no number. The
scores and their source are recorded here so the data is on the record without entering
the prose.

| Dimension                     | Luxembourg score (approx.) |
| ----------------------------- | -------------------------- |
| Power Distance                | 40                         |
| Individualism                 | 60                         |
| Motivation toward Achievement | 50                         |
| Uncertainty Avoidance         | 70                         |
| Long Term Orientation         | 64                         |
| Indulgence                    | 56                         |

Source: Geert Hofstede, Hofstede Insights (the Country Comparison tool and the
six-dimensions model of national culture). Scores are approximate, as published by
Hofstede Insights / The Culture Factor Group.
