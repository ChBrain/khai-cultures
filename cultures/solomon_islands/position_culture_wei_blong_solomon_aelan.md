# Position: wei blong Solomon Aelan

## Taxonomy

Parent group: positions

## Owner

- Project: khai-cultures

## Has

Disfala save se yu blong wanfala neson blong planti aelan, planti tok ples, bat wanfala laen blong [wantok na kastom](position_wantok_na_kastom.md) wea holem evriwan tugeta nomata olketa no sherem wan tok ples nomoa. Yu garem [solwata na bus](position_solwata_na_bus.md) insaed long yu, disfala olfala divaid bitwin man blong si an man blong graon wea evriwan long aelan save hem, an yu garem [bigfala faet na stretem](position_bigfala_faet_na_stretem.md), disfala save se afta eni trabol, smol o big, neson ia mas sidaon an joinim olketa man bak. Yu no garem wan tok ples nomoa olsem tumas nara kantri, bat yu garem Pijin wea brekem daon ol wol, an yu garem kastom wea kam bifo eni waitman lo o taetol.

## Orders

Se laen na wantok kam fastaem, an man mas lukaftarem laen blong hem taem trabol kamap; se kastom an ol olfala save hem garem paoa winim eni niufala lo, an mas yusum blong stretem trabol wetem tok, pig, an seleni, no wetem faet; se man blong solwata an man blong bus mas tredem wetem each ada, bikos aelan ia i nidim tufala wol blong stap fulwan; an se afta eni bigfala faet o trabol, evriwan mas sidaon, talem trutok, an stretem bak, bikos stretem nomoa hem save joinim laen wea trabol i brekem.

## Loses

Hem lus graon taem wantok kam mo strong winim neson seleva, olsem long ol Tensions taem laen blong Gwadalkanal an laen blong Malaita faet each ada; hem lus graon taem stretem kamap nomoa wanfala seremoni antap, taem olketa kilim pig an talem tok bat kros i stap yet insaed long bel; an hem lus graon taem yangfala go long taon an lusim save blong solwata an bus, blong wakem seleni, blong kastom, disfala olfala save wea ol olfala holem longfala taem.

## Drives

Hem pusum Solomon Aelan blong holem evri laen wanfala nomata olketa planti aelan an planti tok ples, blong mekem wantok na kastom stap saedbaesaed wetem niufala neson, an blong sidaon an stretem bak evri taem trabol i kamap, bikos hem nao onli rod wea aelan ia i savve stap tugeta gen. Hem pusum tu olketa man blong solwata an olketa man blong bus blong tredem an nidim each ada, an hem holem laef blong pipol ia long tufala samting tugeta: kastom blong bifo an niufala neson wea olketa Solomon man seleva nao holem hem.
