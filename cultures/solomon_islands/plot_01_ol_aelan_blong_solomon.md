# Plot: ol aelan blong Solomon

## Taxonomy

[play_solomon_islands](play_solomon_islands.md)

## Owner

- Project: khai-cultures

## Cue

Long bifo taem tumas, ol Melanesian pipol kam long si an stap long olketa aelan, mekem olketa difren laen an klosap wan handred tok ples. Olketa solwata pipol wakem laef long lagun an long rif, an olketa bus pipol long insaed graon. Bihaen, long 1568, wan Spanis man kam long si an neimim ples ia bihaen long gol blong Solomon, tingse hem bae faendem bigfala gol, nomata hem no faendem eni gol.

## Action

Longfala taem olketa Melanesian pipol stap long olketa aelan, wakem laef from solwata an from gaden, wakem olketa aelan antap long rif wetem han, an wakem tafuliae, seleni blong sel, wea joinim solwata pipol an bus pipol tru long tred. Bihaen, long 1800s, taem blong leba treid kam: olketa sip kam an tekem awe plande man go waka long plantesin long farawe ples, samfala go from olketa wantem, bat plande go wetem trik an wetem fos. Plande no kam bak. Long disfala taem nao aotsaed wol kam strong antap long olketa aelan, an laef blong olketa Solomon man no semsem moa.

## Stage

- [man blong seleni](persona_man_blong_seleni.md): solwata man wea wakem seleni blong sel, holem olfala save blong solwata pipol.
- [Langa Langa Lagoon](place_langa_langa.md): lagun blong solwata pipol an blong seleni blong sel, ples wea solwata an bus joinim long tred.
- [solwata na bus](position_solwata_na_bus.md): disfala divaid bitwin solwata pipol an bus pipol wea seleni blong sel joinim.

## Tension

Long disfala taem olketa Melanesian pipol stap an wakem wanfala laef wea garem save, tred, an seleni blong olketa seleva, planti laen an planti tok bat joinim long solwata an bus. Bat taem aotsaed wol kam, hem kam wetem hevi: leba treid tekem awe plande man, samfala wetem trik, an putim wanfala skar antap long olketa aelan wea stap kasem tude. Hem nao taem wea aelan ia statem fes long bigfala wol aotsaed.
