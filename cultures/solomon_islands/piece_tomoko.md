# Piece: tomoko

## Taxonomy

Parent group: pieces

## Owner

- Project: khai-cultures

## Place

Long si an long bigfala kastom seremoni: wanfala longfala kanu blong olketa plang wea olketa joinim tugeta, no wanfala kanu wea olketa katem aot long wanfala diwai nomoa, garem hae het long fran an bihaen, an dekoretem wetem sel waet long olraon. Hem nao wanfala bigfala wok blong han blong olketa man blong wes, mo bigwan from Western Province, wea bifo olketa yusum hem blong go long si longwe.

## Load Bearing

Hem holem [bigfala faet na stretem](position_bigfala_faet_na_stretem.md), bikos hem nao wanfala simbol blong taem bifo taem olketa laen faet long si an blong save an strong blong wanfala laen. Wan tomoko hem no waka blong wan man: hem nidim plande man blong wakem hem an plande man blong padol hem tugeta, so hem soem wantok an save blong joinim tugeta. Tude hem no moa kanu blong faet, bat wanfala simbol blong save blong han, blong histri blong olketa laen, an blong wei blong olketa man blong joinim tugeta long wanfala bigfala wok. Sapos no garem disfala kanu, neson ia bae lus wanfala bigfala pat blong save blong wakem an blong histri blong olketa.

## Apparent

Wanfala longfala kanu blong olketa plang wea olketa fasten tugeta, garem hae fran an hae bihaen, dekoretem wetem olketa smol pis sel waet. Long lukluk hem nomoa plang an sel an rop. Bat hevi blong hem hem stap insaed long save blong wakem hem, long plande han wea joinim blong padol hem, an long histri wea hem karem.

## Yearbook

- Olketa man blong Western Province wakem disfala kanu longfala taem bifo, blong go long si longwe.
- Hem nidim plande man blong wakem hem an blong padol hem tugeta, so hem soem wantok an save blong joinim.
- Kasem tude olketa wakem tomoko blong olketa bigfala kastom seremoni, no moa blong faet, bat olsem wanfala simbol blong save an histri.
