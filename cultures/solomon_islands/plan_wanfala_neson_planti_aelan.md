# Plan: wanfala neson, planti aelan

## Taxonomy

Disfala bigfala wok wea neson ia putim long hem seleva: blong holem [wantok na kastom](position_wantok_na_kastom.md) strong nomata planti aelan an planti tok divaidim olketa, blong stretem ol Tensions wea kam from [bigfala faet na stretem](position_bigfala_faet_na_stretem.md) an joinim bak olketa laen, an blong lukaftarem gud [solwata na bus](position_solwata_na_bus.md), olketa lagun an bus an si wea givim aelan ia laef.

## Owner

- Project: khai-cultures

## Direction

Stretem ol Tensions tru kasem rut, no antap nomoa, mekem trutok kam aot an joinim bak olketa laen tru long kot an tru long kastom stretem; holem wantok an neson saedbaesaed, mekem se wantok no kam wanfala chen wea brekem neson; holem evri aelan an evri tok ples insaed long wanfala neson, mekem se no wan laen fil olsem hem stap aotsaed; an lukaftarem gud olketa lagun, rif, an bus, mekem se olketa pikinini wea kam bihaen savve garem si an graon wea givim kaikai yet.

## Orders

- Long man blong evridei olsem [man blong fising](persona_man_blong_fising.md): hol strong long save blong solwata an blong ples blong yu, go aot seif, salem fis, an lukaftarem lagun blong yu blong olketa pikinini.
- Long olketa leada olsem [Sir Peter Kenilorea](persona_kenilorea.md): holem wantok an neson wanfala, no putim wan laen winim narawan, an wakem wanfala neson wea evri laen savve sidaon insaed.
- Long neson: stretem ol Tensions kasem rut, holem wantok strong, an lukaftarem olketa lagun an bus, no fogetem se evri faet mas garem wanfala stretem.

## Implementation

Wanfala neson wea kasem independens long 1978, wea pas tru long ol Tensions an faendem rod blong stretem bak tru long RAMSI, tru long kot, an tru long kastom stretem, an wea holem wantok an kastom strong yet. Bat hem garem had wok yet: blong holem evri aelan an evri laen wanfala, blong mekem wantok no kam wanfala chen, blong joinim laen an laen tru kasem rut, an blong lukaftarem olketa lagun an bus aninit long bigfala wol wea kam strong.
