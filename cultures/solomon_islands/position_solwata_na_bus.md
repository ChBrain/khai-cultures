# Position: solwata na bus

## Taxonomy

Parent group: positions

## Owner

- Project: khai-cultures

## Has

Disfala divaid wea ran tru long evri aelan: long wan saet ol solwata pipol wea stap klosap long si, long olketa lagun, long olketa aelan wea olketa wakem wetem han antap long rif, an wea laef blong olketa kam from fising an from solwata. Long narafala saet ol bus pipol wea stap insaed long graon, antap long maunten, wea laef blong olketa kam from gaden, from yam an kumara an pig. Disfala tufala wol no semsem: kaikai blong olketa difren, save blong olketa difren, an seleni blong sel wea solwata pipol wakem hem nao kam wanfala samting wea joinim tufala wol, bikos solwata pipol salem fis an seleni go long bus pipol blong kasem kaikai blong gaden. Disfala divaid hem olfala olsem aelan ia seleva.

## Orders

Se wan man hem savve hu hem from ples blong hem, sapos hem solwata o bus, an disfala save talem wanem hem kaikai, hao hem waka, an wanem save hem garem; se solwata pipol an bus pipol mas tredem wetem each ada, fis go ap an gaden kaikai kam daon; an se seleni blong sel hem nao rod wea joinim tufala wol an mekem maritim, peimen, an stretem savve hapen.

## Loses

Hem lus graon taem solwata pipol an bus pipol no moa tredem an no moa nidim each ada, taem seleni blong sel kamap nomoa wanfala samting blong tourist, no moa wanfala rod blong joinim olketa. Hem lus graon tu taem laef long taon pulum yangfala awe from ples an from save blong wakem aelan an wakem seleni, kasem disfala olfala save hem klosap lus.

## Drives

Hem stap insaed long man blong seleni, wea long Langa Langa wakem tafuliae wetem han blong hem, an insaed long man blong fising, wea go long lagun blong kasem fis an save evri ples insaed long rif. Hem pusum Solomon Aelan blong holem disfala olfala divaid an disfala olfala join, solwata an bus, wea givim aelan ia kaikai an seleni an wanfala wei blong joinim olketa man wea no semsem.
