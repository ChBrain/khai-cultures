# Play: Solomon Aelan

## Estate

[khai-cultures](https://github.com/ChBrain/khai-cultures): hem nao haos wea holem disfala staging an ansa long hem.

## Name

Solomon Aelan, wanfala chain blong planti aelan long Melanesia long saot-wes Pasifik, long is blong Papua New Guinea an long not blong Vanuatu. ISO SB, languis pis. Hem garem klosap wan handred an samfala tok ples blong olketa difren ples, bat Pijin nao hem tok wea evriwan savve, an wantok hem laen wea holem evri man tugeta.

## Arc

Wanfala neson blong planti aelan an planti tok, bat wanfala laen blong wantok holem olketa. Long bifo taem ol Melanesian pipol kam an stap long olketa aelan, an wan Spanis man neimim ples ia bihaen long gol blong Solomon, nomata hem no faendem eni gol. Long 1800s taem blong leba treid kam, wea olketa man go waka long plantesin long farawe ples, an samfala go wetem trik. Bihaen long datwan Britis Empaea putim protektoret antap long olketa aelan. Long bigfala faet, long Wol Wo Tu, Gwadalkanal kamap ples blong wanfala bigfala faet wea sentem rod blong Pasifik wo, an olketa man olsem Vouza kamap hiro. Bihaen long wo, Maasina Rul kamap, wanfala muvmen blong olketa pipol blong sapotem kastom an askem mo se long olketa seleva. Long 1978 neson ia kasem independens. Aninit long evri samting i garem wanfala divaid: solwata pipol blong solwata an bus pipol blong insaed graon, seleni blong sel an au panpaep blong bambu. Long 1998 go kasem 2003 hem garem Tensions, taem blong trabol, an bihaen long datwan wanfala longfala wok blong stretem evriwan bak tugeta. Aninit long evri samting i garem wanfala nev: wanfala pipol wea isi nomoa, sapotem wantok, lukluk long kastom, an wantem se evriwan i stretem bak tugeta.

## Stakes

Sapos wanfala neson blong planti aelan an planti tok savve stretem bak ol Tensions, holem wantok strong, an lukaftarem gud olketa lagun an bus blong hem, o nogat. Hem garem disfala kwestin: bae olketa pipol blong difren ples savve stap wanfala neson, sherem rod, an pasim disfala graon long olketa pikinini wea bae kam bihaen, o bae olketa samting wea divaidim olketa brekem hem.
