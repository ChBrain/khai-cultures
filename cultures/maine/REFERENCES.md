# Maine: References

The provenance behind this culture: the Hofstede source data that tunes the pitch (kept here, never in the prose) and the public-domain historical sources behind the plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_maine.md](pitch_maine.md) is written from Maine's profile inflected by its lobster fisheries, forest timberlands, and coastal lighthouses, naming no dimension and carrying no number.

| Dimension                     | Maine base score (estimated / US proxy) |
| ----------------------------- | --------------------------------------- |
| Power Distance                | 40                                      |
| Individualism                 | 80                                      |
| Motivation toward Achievement | 64                                      |
| Uncertainty Avoidance         | 44                                      |
| Long Term Orientation         | 26                                      |
| Indulgence                    | 62                                      |

Source: Geert Hofstede, Hofstede Insights (US comparison, adjusted for Maine's low power distance in fishery cooperatives, active local town boards, and low uncertainty avoidance in marine transportation innovation).
