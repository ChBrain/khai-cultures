# Arizona: References

## Hofstede source data

The pitch [pitch_arizona.md](pitch_arizona.md) is written from Arizona's profile inflected by its desert environments, mineral resources, Native American histories, and modern solar expansion, naming no dimension and carrying no number.

| Dimension                     | Arizona base score (estimated / US proxy) |
| ----------------------------- | ----------------------------------------- |
| Power Distance                | 36                                        |
| Individualism                 | 80                                        |
| Motivation toward Achievement | 64                                        |
| Uncertainty Avoidance         | 42                                        |

Source: Geert Hofstede, Hofstede Insights (US comparison, adjusted for Arizona's desert self-reliance, tribal communal networks, and low power distance in local land management).
