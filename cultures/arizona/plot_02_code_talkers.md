# Plot: The Code Talkers Contribution 1942

## Taxonomy

[play_arizona](play_arizona.md)

## Owner

- Project: khai-cultures

## Cue

In 1942, Navajo recruits enter the U.S. Marine Corps, developing an unbreakable communications code based on their native language.

## Action

The code establishes secure communications during critical Pacific battles, launching a legacy of native military service. Yazhi documents the oral histories of code talker families, while Ramon studies the military history of Cochise County.

## Stage

- [navajo weaving](process_navajo_weaving.md) : the traditional pattern-making that represents the structured complexity of the Navajo language.
- [navajo rug](piece_navajo_rug.md) : the hand-woven rug representing native cultural identity and design.
- [Yazhi](persona_yazhi.md) : the cultural guide who honors the language legacy of the code talkers.

## Tension

The code talkers are ordered to keep their work secret after the war, delaying public recognition of their contribution until declassification in 1968.
