# Plan: ရွှေပြည်တော်

## Taxonomy

ဤပြည်က မိမိကိုယ်ကို ချမှတ်ထားသော မပြီးဆုံးနိုင်သေးသော တာဝန်ဖြစ်သည်၊ ခေတ်ပြောင်းကာလများ ဖြတ်သန်းရင်း [ဓမ္မ](position_dhamma.md)ကို နှလုံးသား၏ တရားအဖြစ် ဆက်လက် ထိန်းသိမ်းရန်၊ [ကုသိုလ်](position_merit.md)ကို အရင်းခံသော ဘဝကို မျိုးဆက်သစ်လက်သို့ လက်ဆင့်ကမ်းရန်၊ မျိုးစုံ၊ ဘာသာစကားစုံတို့ဖြင့်လည်း [ပြည်ထောင်စု](position_the_union.md)တစ်ခုအဖြစ် မတူညီမှုအတွင်း တူညီမှုကို ထိန်းသိမ်းရန်၊ ဧရာဝတီမြစ်၊ ပုဂံ၏ စေတီများ၊ စာပေနှင့် ဆိုင်းဝိုင်း၊ ပွဲတော်များကို ထိန်းသိမ်းရန် ဖြစ်သည်။

## Owner

- Project: khai-cultures

## Direction

ပုံသဏ္ဌာန် ပြောင်းသော်လည်း အသက်ဝိညာဉ် မပြောင်းစေရ၊ ရှေးရိုး ဘုရားတရားကို ခေတ်သစ်ဘဝ၌လည်း အသက်ဝင်စေ၊ ကုသိုလ်ကို အရင်းခံသော ဘဝကို မျိုးဆက်သစ်၌လည်း အမြစ်ရှိစေ၊ မျိုးစုံ၏ ထွန်းကားမှုကို ပြည်ထောင်စု၏ ကြွယ်ဝမှုအဖြစ် ထိန်းသိမ်း၊ မြန်မာစကားနှင့် စာပေကို ပြန်လည်နုပျိုစေ၊ ပွဲတော်များနှင့် ရိုးရာ အနုပညာတို့ကို မပျက်စေဘဲ၊ ဤပြည် ခေတ်သစ်သို့ ဝင်ရောက်စဉ်တွင်ပင် မိမိ၏ အမြစ်ကို မမေ့စေရန် ဖြစ်သည်။

## Orders

- ဆရာ၊ မိဘ၊ သံဃာတော်တို့သို့: ဘုရားတရားနှင့် ကုသိုလ်ကို မျိုးဆက်သစ်လက်သို့ မပြတ်တမ်း လက်ဆင့်ကမ်းပေးရန်။
- ကိုရင်လောင်းကဲ့သို့ မျိုးဆက်သစ်သို့: ရိုးရာကို ရင်ထဲ၌ ပိုင်ဆိုင်ကာ၊ ဘုရားတရားနှင့် ကုသိုလ်ကို ဘဝ၏ အရင်းအမြစ်အဖြစ် ထိန်းသိမ်းရန်။
- ပြည်ထောင်စုသို့: မျိုးစုံ၊ ဘာသာစကားစုံတို့၏ မတူညီမှုကို တန်ဖိုးထားကာ၊ မတူညီမှုအတွင်း တူညီမှုကို ဆက်လက် ရှာဖွေရန်။

## Implementation

နှစ်ပေါင်းနှစ်ထောင်နီးပါး ဘုရားတရား စိမ့်ဝင်နေသော ဤရွှေပြည်တော်သည် ခေတ်ပြောင်းကာလများ ဖြတ်သန်းပြီးနောက်လည်း ဘုရားတရား၊ ကုသိုလ်၊ ဧရာဝတီ၊ ပွဲတော်များကို မပျက်စေဘဲ ထိန်းသိမ်းနိုင်ခဲ့သည်၊ သို့သော် ရှေ့တွင် ပိုမိုရှည်လျားသော ခရီး ကျန်ရှိနေသေးသည်၊ ရှေးရိုးနှင့် ခေတ်သစ်ကြား ဟန်ချက်ကို ထိန်းရန်၊ မျိုးစုံ၏ ထွန်းကားမှုကို ပြည်ထောင်စု၏ ကြွယ်ဝမှုအဖြစ် ဆက်လက် ထိန်းသိမ်းရန် ဖြစ်သည်။

## Targets

- [x] ဘုရင်အနော်ရထာ ပုဂံ၌ ထူထောင်ခဲ့သော [ဓမ္မ](position_dhamma.md)သည် ယနေ့တိုင် နေ့စဉ်ဘဝတွင် အသက်ဝင်နေဆဲဖြစ်သည်။
- [x] [ကုသိုလ်](position_merit.md)ကို အရင်းခံသော ဘဝနှင့် ဆွမ်းခံ၊ ရှင်ပြုပွဲ ဓလေ့တို့ ခေတ်အဆက်ဆက် ဆက်လက် တည်ရှိနေသည်။
- [x] မြန်မာစာပေနှင့် ဆိုင်းဝိုင်း၊ ပွဲတော်များ၊ သနပ်ခါးနှင့် လုံချည်တို့ ထိန်းသိမ်းထားနိုင်ခဲ့သည်။
- [W] ရှေးရိုးနှင့် ခေတ်သစ်ကြား ဟန်ချက်ကို ထိန်းကာ၊ ဘုရားတရားကို နှလုံးသား၏ တရားအဖြစ် ဆက်လက် ထိန်းသိမ်းရန်။
- [W] မျိုးစုံ၊ ဘာသာစကားစုံတို့၏ မတူညီမှုကို [ပြည်ထောင်စု](position_the_union.md)၏ ကြွယ်ဝမှုအဖြစ် ဆက်လက် တန်ဖိုးထားရန်။
- [ ] မြန်မာစကားနှင့် စာပေကို မျိုးဆက်သစ်အတွက် ပြန်လည်နုပျိုစေရန်။
- [ ] "ပုံသဏ္ဌာန် ပြောင်းသော်လည်း အသက်ဝိညာဉ် မပြောင်းစေရ" ဟူသော ဆက်ခံရေး တရားကို မျိုးဆက်သစ်တိုင်း၏ လက်သို့ ပေးအပ်ရန်။
