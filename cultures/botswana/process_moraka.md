# Process: go ya kwa morakeng

## Taxonomy

Parent group: processes

## Owner

- Project: khai-cultures

## Initiated by

Losika lo lo nang le dikgomo, le banana ba ba isiwang kwa morakeng go ya go disa: rragwe yo o tlhomamisang gore dikgomo di babalelwa, ngwana yo o ithutang go disa, le monnamogolo yo o tlogelelang bana boswa jwa go disa. Go tlhoka fela moraka, dikgomo, le motho yo o ikemiseditseng go itshokela letsatsi le serame, mme losika lo bo lo tswela kwa nageng go ya go disa.

## Direction

Go ya kwa go itshokeng le go ithuteng boikarabelo: moraka ke sekolo se monna a ithutang go nna a le esi le diphologolo, go itse metsi le bojang, le go disa khumo ya losika ka boinaaketso. Ga se go disa fela mme ke tsela ya go ithuta botho jwa go disa, ngwana a tlogela motse a bo a boa e le motho yo o itseng tiro le boikarabelo.

## Lever

Bo tsamaisiwa ke tumelo e e kwa boteng ya Setswana ya gore dikgomo ke khumo le botshelo, le ke moraka o o dirileng monna mongwe le mongwe modisi wa boswa jwa losika lwa gagwe. Bo ema kgatlhanong le sekaka se se omeletseng le bolwetse jwa dikgomo; kgatlhanong le jone go ema go iphapha ga go nna o le esi le bokgakala jo bo fa gare ga moraka le motse.

## Echo

Go ya kwa morakeng go utlwatsa se se gogang [dikgomo](position_dikgomo.md): gore naga e agilwe mo go diseng le mo boikarabelong jwa losika. Mo morakeng o, [Modisa](persona_modisa.md) o disa dikgomo tsa losika a le esi le sekaka, a boloka khumo le botho jwa losika.
