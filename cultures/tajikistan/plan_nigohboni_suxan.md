# Plan: нигоҳбони сухан

## Taxonomy

Вазифае, ки миллат бар дӯши худ гузоштааст: нигоҳ доштани он чӣ [сухани форсӣ](position_suxani_forsi.md) ба миллат додааст, забон ва шеъри куҳан, нигоҳ доштани зиндагӣ дар [кӯҳҳо](position_kuhho.md) ва водиҳои баланд, ва зинда нигоҳ доштани [меҳмоннавозӣ](position_mehmonnavozi.md) ва оромӣ дар давлати нав.

## Owner

- Project: khai-cultures

## Direction

Забон ва шеъри куҳанро ба насли нав гузаронидан, то ки кӯдак байти ниёгонро аз ёд кунад ва дар ғурбат ҳам онро гум накунад; деҳаи кӯҳиро зинда нигоҳ доштан, то ки кӯҳҳо хомӯш нашаванд; меҳмоннавозӣ ва дастархони фарохро ҳифз кардан; ва оромиро, ки бо иззат ба даст омад, чун амонат нигоҳ доштан ва захмро дубора накушодан.

## Orders

- Ба деҳқон чун [Деҳқон](persona_dehqon.md): деҳаро тарк накун, замин ва дастархонро нигоҳ дор, ва забону шеъри куҳанро ба наберагон гузарон.
- Ба насли нав: байти ниёгонро аз ёд кун, дар ғурбат ҳам забонро гум накун, ва Наврӯзро чун решаи зинда нигоҳ дор.
- Ба миллат: оромиро, ки бо иззат ба даст омад, чун амонат нигоҳ дор, захмро дубора накушо, ва дарро ба рӯи меҳмон кушода нигоҳ дор.

## Implementation

Миллате ҷавон дар давлати худ, ки решаи он ҳазор сол куҳан аст, ки забон ва Наврӯзро дубора устувор кардааст ва аз давраи сахт ба сулҳ роҳ ёфтааст, аммо ҳанӯз бо имтиҳони нигоҳ доштани забон дар ғурбат, зинда нигоҳ доштани деҳаи кӯҳӣ, ва нав нигоҳ доштани оромӣ рӯ ба рӯ аст.

- [x] Решаи куҳан дар асри Сомониён ва шеъри Рӯдакӣ гузошта шуд.

- [x] Истиқлол ба даст омад ва Наврӯз дубора чун ҷашни миллӣ зинда шуд.
- [x] Миллат давраи сахтро бо иззат паси сар кард ва ба сулҳу оштӣ роҳ ёфт.
- [W] Забонро дар миёни наслҳои нав ва дар ғурбат тунук нашудан нигоҳ доштан.
- [W] Деҳаи кӯҳиро зинда нигоҳ доштан, вақте ки ҷавонон ба шаҳру ғурбат мераванд.
- [ ] Наврӯз, меҳмоннавозӣ ва шеъри куҳанро барои ҳар насли нав зинда нигоҳ доштан.
- [ ] Оромиро, ки бо иззат ба даст омад, чун амонат нигоҳ доштан.
