# Uganda: References

The provenance behind this culture: the Hofstede source data that tunes the pitch
(kept here, never in the prose) and the public-domain and standard sources behind
the plots, personas, places, processes, and pieces.

## Hofstede source data

The pitch [pitch_en.md](pitch_en.md) is written from Uganda's region on Geert
Hofstede's model of national culture and names no dimension and carries no number.
The profile and its source are recorded here so the data is on the record without
entering the prose.

| Dimension                     | Uganda / East Africa profile |
| ----------------------------- | ---------------------------- |
| Power Distance                | high                         |
| Individualism                 | low (strongly collectivist)  |
| Motivation toward Achievement | moderate                     |
| Uncertainty Avoidance         | moderate                     |
| Long Term Orientation         | lower                        |
| Indulgence                    | moderate                     |

Source: Geert Hofstede, Hofstede Insights (the six-dimensions model of national
culture and the Country Comparison tool). Uganda is represented in the published
data within the East Africa regional grouping; the profile above is qualitative
and approximate, as published by Hofstede Insights / The Culture Factor Group.
