# Iowa: References

The provenance behind this culture: the Hofstede source data that tunes the pitch (kept here, never in the prose) and the public-domain historical sources behind the plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_iowa.md](pitch_iowa.md) is written from Iowa's profile inflected by its cornfields, state fairgrounds, and writing workshops, naming no dimension and carrying no number.

| Dimension                     | Iowa base score (estimated / US proxy) |
| ----------------------------- | -------------------------------------- |
| Power Distance                | 44                                     |
| Individualism                 | 78                                     |
| Motivation toward Achievement | 68                                     |
| Uncertainty Avoidance         | 44                                     |
| Long Term Orientation         | 24                                     |
| Indulgence                    | 64                                     |

Source: Geert Hofstede, Hofstede Insights (US comparison, adjusted for Iowa's low power distance in farming boards, active community panels, and low uncertainty avoidance in agricultural and literary innovations).
