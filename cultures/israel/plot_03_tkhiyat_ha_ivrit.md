# Plot: תחיית העברית

## Taxonomy

[play_israel](play_israel.md)

## Owner

- Project: khai-cultures

## Cue

בסוף המאה התשע־עשרה עולה ארצה [אליעזר בן־יהודה](persona_ben_yehuda.md), ובלבו מחשבה שרבים חשבו לבלתי אפשרית: להחזיר את העברית, לשון התפילה והספר, לפי בני אדם כלשון דיבור חיה.

## Action

בן־יהודה עושה את ביתו לבית העברי הראשון, מגדל את בנו כילד העברי הראשון, מחבר מילים חדשות וערך מילון גדול. מורים מלמדים עברית בבתי הספר, ילדים משחקים בה בחצר, וכעבור דור היא לשון אם של עם שלם. [חיים נחמן ביאליק](persona_bialik.md) ממלא את הלשון המחודשת בשירה ונותן לה קול ורגש. כך קמה [הלשון המחודשת](position_ha_lashon.md): לשון קודש ששבה לחול, מקרה כמעט יחיד בתולדות הלשונות.

## Stage

- [אליעזר בן־יהודה](persona_ben_yehuda.md): מי שהחזיר את העברית לפי בני אדם.
- [חיים נחמן ביאליק](persona_bialik.md): המשורר שמילא את הלשון המחודשת בשירה.
- [הלשון המחודשת](position_ha_lashon.md): לשון הקודש ששבה והיתה לשון דיבור חיה.
- [תל אביב](place_tel_aviv.md): העיר העברית החדשה שבה נשמעה הלשון ברחוב ובשוק.

## Tension

הלשון שבה לחיים, אך עִמה שבה גם השאלה הישנה: האם החול יבלע את הקודש, ואם אותן מילים ששימשו לתפילה ולנבואה יישאו עוד את עומקן כשהן נאמרות בשוק ובחצר. בתחייה גלום פלא וגם אחריות: לדבר את הלשון העתיקה בלי לרוקנה מזכרונה.
