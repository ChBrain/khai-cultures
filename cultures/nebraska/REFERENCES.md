# Nebraska: References

The provenance behind this culture: the Hofstede source data that tunes the pitch (kept here, never in the prose) and the public-domain historical sources behind the plots, personas, places, and pieces.

## Hofstede source data

The pitch [pitch_nebraska.md](pitch_nebraska.md) is written from Nebraska's profile inflected by its cattle ranches, prairie woodlands, and historic pioneer trails, naming no dimension and carrying no number.

| Dimension                     | Nebraska base score (estimated / US proxy) |
| ----------------------------- | ------------------------------------------ |
| Power Distance                | 44                                         |
| Individualism                 | 78                                         |
| Motivation toward Achievement | 68                                         |
| Uncertainty Avoidance         | 44                                         |
| Long Term Orientation         | 24                                         |
| Indulgence                    | 64                                         |

Source: Geert Hofstede, Hofstede Insights (US comparison, adjusted for Nebraska's low power distance in ranch crews, active community panels, and low uncertainty avoidance in windbreak forestry innovation).
