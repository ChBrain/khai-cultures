# Colorado: References

## Hofstede source data

The pitch [pitch_colorado.md](pitch_colorado.md) is written from Colorado's profile inflected by its gold rush settlement, alpine environment, clean-tech industries, and outdoor recreation focus, naming no dimension and carrying no number.

| Dimension                     | Colorado base score (estimated / US proxy) |
| ----------------------------- | ------------------------------------------ |
| Power Distance                | 32                                         |
| Individualism                 | 88                                         |
| Motivation toward Achievement | 60                                         |
| Uncertainty Avoidance         | 44                                         |
| Long Term Orientation         | 28                                         |
| Indulgence                    | 68                                         |

Source: Geert Hofstede, Hofstede Insights (US comparison, adjusted for Colorado's flat outdoor industry teams, environmental stewardship, and low power distance in start-up ventures).
